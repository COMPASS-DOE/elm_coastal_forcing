0.1 Ben’s workflow for synoptic runs on groundwater wells

0.1.1 Build hydrological boundary conditions from L2 data

Fetch groundwater water level and salinity from COMPASS L2 releases (gap-filled), aggregate to hourly, and write outputs to: - COMPASS_hydro_BC_fromL2

0.2 Modify domain and surface files

Generate modified domain and surface datasets: - domain_synoptic_OLMT.nc → COMPASS_domain_multicell_fromgw.nc - surfdata_synoptic_OLMT.nc → COMPASS_surfdata_fromgw.nc

0.2.1 Surface-data edits for groundwater-well forcing

When supplying water levels relative to the land surface, set height/stream-distance fields consistently:

1 Set height above stream to 0 since water levels are supplied relative to surface

surfdata_multicell["ht_above_stream"] = 0.0 * surfdata_multicell["TOPO"] surfdata_multicell["dist_from_stream"] = 1.0 + 0.0 * surfdata_multicell["ht_above_stream"]

surfdata_multicell.to_netcdf(“COMPASS_surfdata_fromgw.nc”) domain_multicell.to_netcdf(“COMPASS_domain_multicell_fromgw.nc”)

If using the groundwater wells, set the topo and height to zero.

Set height above stream to 0 since water levels are being supplied relative to surface surfdata_multicell['ht_above_stream'] = surfdata_multicell['TOPO']*0.0 surfdata_multicell['dist_from_stream'] = surfdata_multicell['ht_above_stream']*0.0+1.0 `

surfdata_multicell.to_netcdf(‘COMPASS_surfdata_fromgw.nc’) domain_multicell.to_netcdf(‘COMPASS_domain_multicell_fromgw.nc’) `

1.0.1 Fdrain:

fdrain controls the relationship between water table depth and topographic drainage In surface data file: surfdata_multicell['fdrain']=surfdata_multicell['dist_from_stream']*0.0 + 2.5
